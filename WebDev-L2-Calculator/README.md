# Calculator

A sleek, dark-themed premium calculator built with vanilla web technologies. It features floating background orbs, a glassmorphic card design, hover ripple effects, and a history expression display.

## Features
- **Modern UI:** Glassmorphism card, custom Google Fonts (Inter), and glowing buttons.
- **Micro-interactions:** Interactive hover effects, hover ripples that track the mouse cursor, and button press animations.
- **Dynamic Display:** Shows both the current input/result and the mathematical expression history above it.
- **Responsive Layout:** Adapts cleanly to mobile and desktop viewports using CSS Grid and Flexbox.
- **Robust Logic:** Supports chained operations (e.g., `5 + 3 * 2` evaluates correctly step-by-step) and displays an error message for division by zero instead of crashing.
- **Vanilla Tech:** Zero dependencies, libraries, or frameworks.

## Technologies Used
- HTML5
- CSS3 (Variables, Grid, Flexbox, Keyframe Animations)
- JavaScript (Vanilla DOM APIs and Event Listeners)

## How to Run the Project
1. Clone or download this folder to your local computer.
2. Locate the `index.html` file in your file explorer.
3. Double-click `index.html` (or right-click and choose **Open with**) to run it in any web browser (Chrome, Edge, Safari, Firefox). No servers or installations are required!
