# Tribute Page

A sleek, dark-themed tribute page dedicated to Mahendra Singh Dhoni (Captain Cool) built with vanilla web technologies. It features a modern layout, an alternating career timeline, counter statistics, and responsive grid achievements.

## Features
- **Modern UI:** Minimal dark luxury theme, custom Google Fonts (Cinzel & Poppins), and subtle glowing accent buttons.
- **Micro-interactions:** Scroll-reveal animations, interactive hover scaling on cards, and button click micro-movements.
- **Intersection-Observed Stats:** Triggers a counting animation automatically when statistics scroll into the viewport.
- **Alternating Timeline:** A clean vertical timeline displaying Dhoni's career milestones from his 2004 debut to the 2023 IPL title.
- **Responsive Layout:** Adapts cleanly to mobile, tablet, and desktop viewports using CSS Grid and Flexbox.
- **Vanilla Tech:** Zero dependencies, libraries, or external frameworks.

## Technologies Used
- HTML5 (Semantic Structure & SEO Metadata)
- CSS3 (Variables, Grid, Flexbox, Keyframe Animations)
- JavaScript (Vanilla DOM APIs, Intersection Observer)

## How to Run the Project
1. Clone or download this folder to your local computer.
2. Locate the `index.html` file in your file explorer.
3. Double-click `index.html` (or right-click and choose **Open with**) to run it in any web browser (Chrome, Edge, Safari, Firefox). No servers or installations are required!
